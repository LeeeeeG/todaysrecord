2-1 컬럼 너비와 높이 지정하기

grid-template-columns 속성을 이용하여 첫번째 컬럼(열)은 100px, 마지막 컬럼 100px 나머지 공간은 자동으로 사용하도록 한다.

grid-template-rows 속성을 이용하여 세로 row(행)를 50px 씩 두개 사용하도록한다.

[ CSS ]

.container{

    display:grid;

   grid-template-columns: 100px auto 100px;

   grid-template-rows: 50px 50px;

}

[ 브라우저 ]

2-2 그리드 간격 지정하기

각 그리드 간의 간격을 3px를 지정한다.

[ CSS ]

.container{

            display:grid;

            grid-template-columns: 100px auto 100px;

            grid-template-rows: 50px 50px;

            grid-gap:3px;

}

[ 브라우저 ]

첫 column은 100px 나머지는 자동으로,

첫번째 row(행)은 50px, 두번째 row은 50px 세번째 row는 200px의 높이를 사용하도록하낟.

[ CSS ]

.container{

            display:grid;

            grid-template-columns: 100px auto;

            grid-template-rows: 50px 50px 200px;

            grid-gap:3px;

}

[ 브라우저 ]

  1. Auto-fit, minmax

https://codepen.io/alikerock/pen/GBOybL

현재 100px 너비의 컬럼(열, column)이 6개 100px 높의 행(row)이 2개인 상태인데, 반응형은 안되어 있다. 너비를 늘리거나 줄여도 크기는 그대로인 상태이다.

[ 브라우저 ]

컬럼의 단위를 아래와 같이 비율대로 유지하도록 1fr를 사용한다.

[ CSS ]

.container {

        display: grid;

        grid-gap: 5px;

        grid-template-columns: repeat(6, 1fr);

        grid-template-rows: repeat(2, 100px);

}

[ 브라우저 ]

6개의 컬럼수를 유지하면서 늘어나거나 줄어들고 있다.

[ 브라우저 ]

컬럼의 너비 100px을 유지하면서 너비에 따라 컬럼의 개수가 늘어나도록 아래와 같이 작성해본다.

[ CSS ]

.container {

        display: grid;

        grid-gap: 5px;

        grid-template-columns: repeat(auto-fit, 100px);

        grid-template-rows: repeat(2, 100px);

}

너비에 따라서 컬럼의 개수가 늘어나거나 줄어들고 있다.

[ 브라우저 ]

그런데 너비를 줄이다 보면 아래와 같이 오른쪽에 공간이 남게 되는 부분이 생긴다.

[ 브라우저 ]

위와 같은 현상을 막기위해 비율대로 컬럼의 너비를 지정하기 위해 아래와 같이 작성해본다.

[ CSS ]

.container {

        display: grid;

        grid-gap: 5px;

        grid-template-columns: repeat(auto-fit, 1fr);

        grid-template-rows: repeat(2, 100px);

}

그러면 1fr에 비율대로 auto-fit으로 채우라고 했기때문에 아래와 같이 브라우저에 나온다.

[ 브라우저 ]

위와 같은 현상을 막기위해 비율대로 컬럼의 너비를 지정하기 위해 아래와 같이 작성해본다.

[ CSS ]

.container {

        display: grid;

        grid-gap: 5px;

        grid-template-columns: repeat(auto-fit, minmax(100px, 1fr));

        grid-template-rows: repeat(2, 100px);

}

컬럼의 최소 너비를 100px을 하되 좌우를 채우는 레이아웃이 완성된다. 100px이하로 줄어들면 다음 줄로 넘어간다.

minmax(100px, 1fr)은 최소값을 100px이고 최대값은 1fr로 공간안에서 비율대로 유지하라는 말.

[ 브라우저 ]

  1. Implicit rows (절대 줄)

https://codepen.io/alikerock/pen/QBOQKK

현재 화면에서 column의 너비는 자연스럽에 100px을 유지하면서 채워지고 있지만 너비가 좁을때는 세로 row의 높이가 일정하지 않은 문제가 있다.

css에서 grid-template-rows: repeat(2, 100px); 를 통해  행(row) 2개의 높이만 지정했기 때문이다.

[ 브라우저 ]

row의 높이를 자동으로 100px로 맞추기 위해 아래와 같이 입력한다.

[ CSS ]

.container {

        display: grid;

        grid-gap: 5px;

        grid-template-columns: repeat(auto-fill, minmax(100px, 1fr));

        grid-auto-rows: 100px;

}

그러면 컬럼개수도 너비에 따라 유동적으로, 높이도 100px를 유지하면서 배열이 된다.

중략…

현재 문제는 아래와 같이 빈 공간이 생긴다는 것이다. 무조건 첫번째 컬럼 라인에서 2개 컬럼 라인을 더한 3번째 컬럼 라인까지를 지정했기 때문이다.

[ 브라우저 ]

시작하는 컬럼라인을 자동으로 두어 아래와 같이 수정한다. 그러면 시작하는 컬럼을 자동으로 두었기때문에 그 컬럼 라인이 어디든지 상관없이 그곳부터 2컬럼 더한 컬럼까지의 너비를 사용하도록 된다.

중략…

}

grid-column: auto / span 2; == grid-column:  span 2;

[ CSS ]

중략…

.horizontal {

        grid-column:  span 2;

}

세로로 긴 이미지를 위해 아래와 같이 입력한다.

[ CSS ]

중략…

        grid-row: span 2;

}

[ 브라우저 ]

큰이미지 클래스명 big을 가지고 있는 이미지는 가로 2컬럼 세로 2컬럼을 사용하도록 아래와 같이 입력한다.

[ CSS ]

중략…

.big {

        grid-column: span 2;

        grid-row: span 2;

}

[ 브라우저 ]

그런데 컬럼이 비어있는 부분이 아래와 같이 있다. 6번째 칸이 비어있는 부분을 보면 해당 자리는 6번째 이미지가 6번자리에 오기에는 너무 크기 때문이다. 그리고 2번째 이미지가 세로로 길어 가리고 있기 때문에 이동을 못한 상태이다.

[ 브라우저 ]

세로로 긴 이미지를 위해 아래와 같이 입력한다. grid-auto-flow:의 값을 dense로 변경하면 앞부분의 빈공간을 채우면서 레이아웃이 완성된다.  너비 또는 높이가 변경되더라도 레이아웃이 자동으로 빈구멍(hole)을 채우면서 완성된다.

참조: https://www.w3schools.com/cssref/pr_grid-auto-flow.asp

[ CSS ]

.container {

        display: grid;

        grid-gap: 5px;

        grid-template-columns: repeat(auto-fit, minmax(100px, 1fr));

        grid-auto-rows: 75px;
