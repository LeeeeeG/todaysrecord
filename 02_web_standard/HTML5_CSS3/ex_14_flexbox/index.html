item 정렬 순서 (자녀요소에 적용)

display:flex / order:상수;

order : 정렬 순서를 변경, 기본값은 0이며 아무것도 적지 않으면 기본값이 적용되고 0이 아닌 다른 값(상수)을 가진 자녀요소는 순서가 변경됨, 값이 커질수록 뒤에 배치

1
2
3

display:flex / 속성표기법 : flex-grow, flex-shrink, flex-basis

flex-grow : 부모요소의 값에 맞게 자녀요소의 비율을 변경해줌, 기존에 지정된 넓이/높이값을 무시함, 자녀요소가 부모요소보다 작을 때는 문제 없으나 부모요소가 더 작을 경우 자녀요소의 크기도 같이 줄어든다, 기본값은 0
flex-shrink : 줄어들 때 2배로 줄어듦, 기본값은 1
flex-basis : 자녀요소의 비율을 지정 한다, 기본값은 auto, auto로 지정할 경우 부모요소의 값에 따라 자동으로 변경되며 자녀요소에 적용되었던 높이와 넓이를 고려함
flex : 0 1 auto 속성표기법으로 지정 가능
flex-basis를 가장 우선적으로 적용
